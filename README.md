## 🌟 Descripción

Pokédex Neo es una aplicación web moderna desarrollada con React y Vite que permite explorar el universo Pokémon a través de una experiencia visual inmersiva y dinámica.

La aplicación consume información en tiempo real desde PokéAPI v2 y presenta cada Pokémon mediante tarjetas animadas, imágenes de alta calidad, efectos visuales fluidos y una interfaz elegante basada en una paleta de tonos oscuros, violetas y acentos brillantes.

Cada tipo de Pokémon posee una identidad visual única, haciendo que colores, efectos y detalles cambien dinámicamente para ofrecer una experiencia más atractiva y personalizada.

## ✨ Características

- ⚡ Datos obtenidos en tiempo real desde PokéAPI v2.
- 🎨 Tarjetas dinámicas con animaciones suaves.
- 🌙 Interfaz moderna con temática oscura.
- 🔍 Búsqueda rápida de Pokémon.
- 🖼️ Imágenes de alta calidad.
- 📱 Diseño responsive para móvil y escritorio.
- 🚀 Alto rendimiento gracias a Vite.
- 🎭 Colores adaptativos según el tipo de Pokémon.

## 🛠️ Tecnologías

- React
- TypeScript
- Vite
- CSS3
- PokéAPI v2

## 🌐 API Utilizada

https://pokeapi.co/
